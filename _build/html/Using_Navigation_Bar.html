Using Navigation BarÂ¶

The Navigation Bar is located at the top of each graph and comprises various functions including saving graphs, zooming, and moving.

# The Navigation Bar used in this program is from NavigationToolbar2Tk, for more information click here.

Features:

  1. Save Figure
    This function saves a picture of the current plot in various formats depending on user selection.

  2. Configure subplots
    Creates a new tab with sliders for adjusting plot layout.

  3. Zoom (Zoom to rectangle x/y fixes axis)
    When selected, allows the user to highlight areas of the plot to zoom in, click on the button again to exit select and use the back or home button to redo changes.

  4. Move/drag button
    Select to move perspective within the plot.

  5. Redo, Undo, reset default Back to the previous view(redo), Forward to the next view(undo), and Reset Original view(home button).
