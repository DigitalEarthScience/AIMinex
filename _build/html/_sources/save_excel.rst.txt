How to Save Data into Excel File
==================================
1. Save PCA Excel Data without Clusterings
--------------------------------------------
    PCA Data for PC values by Samples and PC values by Elements can be saved directly from the File menu after applying the Lithology, Scaler, and PCA component selection.  

    i) Under File Menu open Save PC and Select Data Type
        .. image:: mineralAI_images/document/Saveexcel1.png
            :width: 100%
\
    ii) Save the File
         .. image:: mineralAI_images/document/Saveexcel2.png
             :width: 100%
\

2. Save PCA Excel Data with Clustering
--------------------------------------
    To Save Excel Data with coloured cells based on their clusterings, the cluster graph needs to be plotted first to generate the data needed for Excel.
    
    i) Navigate to any cluster graphs
        .. image:: mineralAI_images/document/savec1.png
            :width: 100%
\
    ii) Set the desired parameters and plot the graph
         .. image:: mineralAI_images/document/savec2.png
             :width: 100%
\
    iii) Save the Excel file using the Save Excel Button
          .. image:: mineralAI_images/document/savec3.png
              :width: 100%
    \
    Excel:
        .. image:: mineralAI_images/document/savec4.png
            :width: 100%
