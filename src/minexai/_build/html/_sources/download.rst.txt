How to Install
===============
1. Pip install MinexAI from PyPI (Not available yet)

2.  
    a. Download MinexAI package
    b. In the terminal, navigate to the MinexAI folder (run "cd MinexAI")
    c. Run "pip install dist/MinexAI-0.0.1-py3-none-any.whl" or "Pip install ."
    d. Type "minexai" to run
