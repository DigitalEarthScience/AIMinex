Using Navigation Bar
====================
| The Navigation Bar is located at the top of each graph and comprises various functions including saving graphs, zooming, and moving.
\
# The Navigation Bar used in this program is from NavigationToolbar2Tk, for more information click `here <https://github.com/matplotlib/matplotlib/blob/main/lib/matplotlib/backends/_backend_tk.py>`_.

Features:
 1. Save Figure |br|
    This function saves a picture of the current plot in various formats depending on user selection.

    .. image:: mineralAI_images/document/nb1.png
       :width: 100%
    \
 2. Configure subplots |br|
    Creates a new tab with sliders for adjusting plot layout.

    .. image:: mineralAI_images/document/nb3.png
        :width: 100%  

    .. image:: mineralAI_images/document/nb2.png
        :width: 100%  
    \
 3. Zoom (Zoom to rectangle x/y fixes axis) |br|
    When selected, allows the user to highlight areas of the plot to zoom in, click on the button again to exit select and use the back or home button to redo changes.         
    
    .. image:: mineralAI_images/document/nb4.png
        :width: 100%  
    \
 4. Move/drag button |br|
    Select to move perspective within the plot.
    
    .. image:: mineralAI_images/document/nb5.png
        :width: 100%  
    \
 5. Redo, Undo, reset default
    Back to the previous view(redo), Forward to the next view(undo), and Reset Original view(home button).   

    .. image:: mineralAI_images/document/nb6.png
        :width: 100%  
    \
